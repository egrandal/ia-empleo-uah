## Paso 1. Usamos Gemini, NotebookLM, Qwen Studio, ChatGPT, para realizar una "investigación" sobre la creación de instrucciones para asistentes de IA

**Objetivo:**
Conseguir una base de conocimiento actualizada de las mejores prácticas para la generación de prompts e instrucciones personalizadas eficientes para asistentes de IA.

---

**Prompt para Deep Research** (*Gemini*):

Investiga las mejores prácticas para crear instrucciones personalizadas para asistentes de inteligencia artificial.
Prioriza documentación oficial de proveedores top de modelos de IA, guías técnicas de prompting y recursos publicados por plataformas de IA reconocidas y líderes en el mercado, como OpenAI, Anthropic, Google, etc...
Necesito datos claros y concisos, estructuras reutilizables, errores a evitar y ejemplos aplicables para diseñar asistentes de alto nivel para usarlo como base de generación de nuevos prompts personalizables.
