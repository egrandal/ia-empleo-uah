# 🤖 IA para la búsqueda de empleo  
### Webinar – Universidad de Alcalá de Henares

Bienvenido/a 👋

Este repositorio contiene todos los materiales prácticos del webinar sobre cómo usar Inteligencia Artificial para mejorar tu empleabilidad.

⚠️ No es teoría. Es un sistema que puedes aplicar desde ya.

---

# 🧠 ¿Qué vas a encontrar aquí?

Este repositorio recoge el proceso completo que vimos en el webinar:

1. Analizar ofertas reales de empleo  
2. Entender qué busca el mercado  
3. Construir tu perfil profesional  
4. Detectar qué te falta (DAFO)  
5. Mejorar tu CV, LinkedIn y estrategia  

Todo usando IA como herramienta.

# 🧩 ¿Qué necesitas para usar esto?

Muy poco:

- Un navegador
- Una cuenta en alguna IA (ChatGPT, Gemini…)
- Ganas de mejorar tu perfil (esto ya es más complicado)

---

# ⚠️ Cosas importantes (léelo, en serio)

- La IA no hace el trabajo por ti  
- Si no tienes criterio, la IA amplifica eso  
- Copiar sin entender = mala idea  

La IA no piensa.  
Predice.

---

# 💡 Cómo sacarle partido de verdad

No intentes hacerlo todo a la vez.

Haz esto:

1. Analiza 3 ofertas reales  
2. Genera perfil objetivo  
3. Compara contigo  
4. Detecta 2–3 mejoras  
5. Aplica  

Y repite.
